## 视频页布局对齐修复

**问题**：video.vue 两处布局偏差：
1. UP主信息栏（`.strip-up`）与下方侧栏推荐视频面板（`.video-side-dock`）左边缘不对齐
2. 整体网格左右不对称（`bili-wrapper` 全局 `padding:0 60px`，侧栏 absolute 向右溢出）

**根因**：
- `.video-info-strip` 用了 `justify-content:space-between`，UP卡被推到 wrapper 最右端（左边缘 ≈722px）
- `.video-side-dock` 是 `position:absolute; left:calc(100%+16px)` 在 `body-stack`(819px) 内 → 左边缘 = 835px
- 两者差 113px；且 60px padding + 60px overflow 导致视觉重心偏移

**修复**（`src/pages/video/video.vue` 样式段）：
- `.video-page` 内嵌套覆盖 `.video-main.bili-wrapper`，设 `padding:0 !important`
- 让 `body-stack(819px) + gap(16px) + side-dock(325px) = 1160px` 刚好填满 wrapper 居中对称
- `.video-info-strip` 去掉 `justify-content:space-between`，改用 `gap:16px`
- `.strip-video`：`flex:1` → `flex:0 0 $video-col-width`（819px）
- `.strip-up.up-card`：宽度 `318px` → `325px`（与 side-dock 同宽）
- `$video-col-width` 变量上移到使用前，删除原处重复定义
