## 一、用户端前台（10 个）

### 1. 用户社交体系（关注/粉丝/订阅/个人主页）
- **状态**：✅ **已实现**。`follow_group.go` / `SpaceRelations.vue` / `PersonalSpace.vue` / `MbSpaceHeaderActions.vue`
- **可改进**：粉丝榜单、关注推送流
- **优先级**：P3（维护）

### 2. 播放器高级功能（多P/章节/倍速/画中画/多码率/进度记录）
- **状态**：🟡 **基础有，进阶无**
  - 进度记录：`view_history.go` + `ViewHistory.vue` 已有
  - 倍速/画中画：HTML5 `<video>` 原生支持，但未在前端显式暴露 UI
  - 多P/章节/多码率：**未实现**（当前单视频单码率 H.264 MP4）
- **阻塞点**：多码率需要 FFmpeg 转码任务切分 HLS + m3u8 切片 + 至少 3 档码率输出
- **优先级**：P1（用户体验提升最直接）

### 3. 字幕管理（上传/编辑/多语种/时间轴/自动转写）
- **状态**：❌ **未实现**
- **依赖**：字幕轨道播放器扩展、Whisper/ASR 集成
- **优先级**：P2

### 4. 评论区增强（楼中楼/置顶/富文本/表情/图片/排序/过滤/举报）
- **状态**：🟡 **基础 3 级评论已实现**（F8），点赞通知聚合已实现（S-010）
  - 缺失：置顶/富文本/表情/图片/排序/过滤
- **优先级**：P1（运营需求高）

### 5. 投稿与创作者中心（编辑/草稿/分P管理/定时发布/稿件统计）
- **状态**：🟡 **投稿/上传/统计基础有**
  - `Upload.vue` + `CreatorShell.vue` + `CmVideoPicker.vue`
  - 缺失：草稿箱、定时发布、分P管理、稿件统计 Dashboard
- **优先级**：P1

### 6. 直播前端（直播间/连麦/观众列表/回放）
- **状态**：❌ **未实现**（SPEC 1.2 明确排除）
- **依赖**：流媒体服务器（SRS/ZLMediaKit）、WebRTC/FLV、连麦 SFU
- **工作量**：3-4 周
- **优先级**：P3（高成本低频）

### 7. 个性化推荐与首页 Feed
- **状态**：🟡 **首页已有**（`/pages/home/`），但**无推荐算法**（SPEC 1.2 排除）
  - `VideoFeed.vue` + `home/videoFeed/`
  - 缺失：协同过滤/向量召回/订阅流/排行榜
- **优先级**：P2

### 8. 通知与消息中心
- **状态**：✅ **已实现**（F9 + S-010 + AC-12）
  - `Messages.vue` 独立页面 + 5 个分类（回复/@/赞/系统/我的）
  - 私信：`dm.go` + `Messages.vue`（mb-chat）
- **优先级**：P3（已完善）

### 9. 活动/专题与运营位
- **状态**：✅ **已实现**
  - `home_banner.go` + `BannerManage.vue` + `popularize.vue`
  - `HotSearchManage.vue` 热搜干预
- **优先级**：P3

### 10. 移动端/小程序适配
- **状态**：🟡 **响应式有**，但**无独立 H5/小程序**
  - `MbDynVideoCommentPanel.vue` 等 mb 前缀组件是移动端样式
  - 缺失：小程序（uniapp / Taro）
- **优先级**：P3
